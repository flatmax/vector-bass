# \<vector-synth\>

Web component for an example WASM C++ repo https://github.com/flatmax/WASM

## Example instantiation of this element (from demo/index.html) :

```
<vector-synth></vecotr-synth>
```

## To run the Demo

### Install modules

```
npm i
```

### Viewing Your Element

```
$ polymer serve
```
